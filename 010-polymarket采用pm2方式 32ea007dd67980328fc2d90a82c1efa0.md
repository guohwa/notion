# 010-polymarket采用pm2方式

**常用管理命令速查表**

| 查看运行状态 | pm2 list |  |
| --- | --- | --- |
| 查看实时日志 | pm2 logs poly-v77 | pm2 logs poly-logic |
| 停止程序 | pm2 stop poly-v77 |  |
| 重启程序 | pm2 restart poly-v77 | pm2 restart poly-logic |
| 删除进程记录 | pm2 delete poly-v77 |  |
| 仪表盘 |  | pm2 monit |

### 部署（更新）方法：

1. 覆盖 `paper_trader.py`
2. 执行 `pm2 restart poly-logic`
3. 查看日志：`pm2 logs poly-logic`

cd /root/poly_bot

第一个脚本  paper_trader.py

cd /root/poly_bot
pm2 delete poly-logic
pm2 start paper_trader.py --name "poly-logic" --interpreter /root/.pyenv/versions/3.10.14/bin/python
pm2 logs poly-logic
